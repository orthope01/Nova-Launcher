# Nova Launcher v3.0.0

Minecraft лаунчер, объединяющий Nova Launcher с логикой **Legacy Launcher**.

## Ключевые исправления (по сравнению с v2.0)

### 🔧 Исправление демо-режима (главная проблема)
- **`is_demo_user` = ВСЕГДА `false`** — точно как в Legacy Launcher
- Правильная обработка `features` в `rules` аргументов Minecraft
- Аргумент `--demo` **никогда** не передаётся в игру
- Оффлайн и Microsoft аккаунты запускают полную игру

### 🌐 Подключение к серверам (как в Legacy Launcher)
- **quickPlay** для Minecraft 1.20+ (через features `is_quick_play_multiplayer`)
- **`--server` / `--port`** для всех версий начиная с 1.3
- Автоматический выбор метода в зависимости от версии
- Поле ввода адреса прямо на главной странице

### 🔐 Microsoft Auth (полный flow)
- Полная цепочка: MS Token → Xbox Live → XSTS → Minecraft → Profile
- Корректная обработка ошибок (нет Xbox аккаунта, детский аккаунт, нет лицензии)
- Детские аккаунты — понятное сообщение об ошибке

### 🎨 Адаптивный интерфейс
- Все размеры через `clamp()` — подстраиваются под размер окна
- Sidebar сворачивается до иконок при маленьком окне
- `minWidth: 750px` — работает даже в небольшом окне

### 📦 UUID совместимость
- Оффлайн UUID: `MD5("OfflinePlayer:<username>")` — Bukkit-совместимый
- Серверы в `online-mode=false` правильно распознают игрока

## Установка

```bash
npm install
npm start
```

## Сборка

```bash
npm run build
```

## Требования
- Node.js 18+
- Java 8, 11, 17 или 21 (в зависимости от версии Minecraft)
